# UI Showcase

Dieser Bereich enthaelt konkrete UI-Bausteine und Interaktionsmuster mit Fokus auf Wiederverwendbarkeit.

Die Themen in diesem Ordner sind naeher an der sichtbaren Produktoberflaeche und sollen zeigen, wie sich haeufige Anforderungen sauber lösen lassen.

## Getting Started

`npm install`

`npm run storybook`

## Übersicht

### Table

Tabellenkomponenten mit Sortierung, Filterung, Statusdarstellung, leeren Zustaenden und moeglichen Erweiterungspunkten fuer unterschiedliche Produktkontexte.

### Calendar

Kalender- und Terminmuster, etwa fuer Auswahl, Anzeige oder Interaktion mit datengetriebenen Oberflaechen.

### Context Menu

Flexible Kontextmenues mit Fokus auf Bedienbarkeit, Positionierung, Tastatursteuerung und Einbettung in komplexere UI-Situationen.

### Filter Chips

Kompakte Filter-Interaktionen fuer Suche, Selektion und Zustandskommunikation in datenlastigen Oberflaechen.
