Fundamentals

Diese Seite ist als einfacher Einstieg fuer Basisbeispiele gedacht. Sie funktioniert als reine statische Seite und eignet sich direkt fuer GitHub Pages.

HTML

Semantik, Struktur und robuste Grundbausteine ohne Framework-Abhaengigkeit.

CSS

Patterns fuer Layout, Theming und wartbare Styling-Konventionen.

JavaScript

Kleine, nachvollziehbare Skripte mit relativen Asset-Pfaden fuer statisches Hosting.

Deploy-Hinweis

GitHub Pages kann diese Inhalte direkt ausliefern, sobald der Ordner in das Pages-Artefakt kopiert wird.
